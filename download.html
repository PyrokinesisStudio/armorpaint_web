Get Preview

DRAFT ONLY.

ArmorPaint is in early stages of development. You can support the project now and get the preview - careful though, this early build has many rough edges and the experience may be frustrating! Everyone accessing the preview version will receive all future releases for free.

ArmorPaint is a stand-alone application for Windows, Linux and MacOS - currently distributed alongside the Armory Engine. ArmorPaint does all the heavy lifting fully on the GPU. For 4K texture painting, Intel HD4000 graphics card or better is recommended. For 20K texture painting, NVidia GTX1060 graphics card or better is recommended.

Please note that eventually, as the project matures, we would love making it free. The funds are (unfortunately!) needed to actively develop project.

Source code is available at GitHub (requires Armory SDK).
