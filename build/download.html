ArmorPaint is in early stages of development. You can support the project and get it now - careful though, this early build has many rough edges and the experience may be frustrating! Before purchasing, please read the manual first to know what parts need more work. Everyone accessing the early versions will receive all future releases for free.

ArmorPaint is a stand-alone application for Windows, Linux and MacOS - currently distributed alongside the Armory Engine. ArmorPaint does all the heavy lifting fully on the GPU. For 4K texture painting, Intel HD4000 graphics card or better is recommended. For 20K texture painting, 6GB graphics card (NVidia GTX1060) or better is recommended.
